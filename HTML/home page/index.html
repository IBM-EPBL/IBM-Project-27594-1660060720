Handwritten Recognition System

Handwritten text recognition is a technology that is much needed in the world of today. This digit recignition system is used to recognize the digits from different sources like emails, bank cheque, papers, images etc. Before proper implementation of this technology we have relied on writing texts with our own hands which can result in errors. It is difficult to store and access physical data with efficiency. The project presence recognizing the hand written digits (0 to 9) from the famous MNIST DATASHEET. Here we will be using Artificial Neural Networks and Convolution Neural Network.
